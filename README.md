## üíª Developer Setup

1. Clone repository:

   ```bash
   git clone https://github.com/KMV531/nextjs-dashborad.git

   ```

2. Install dependencies:

   ```bash
   npm install

   ```

3. Set up environment variables (example):

   ```bash
   POSTGRES_URL= ********
   POSTGRES_URL_NON_POOLING= ********
   POSTGRES_USER= ********
   POSTGRES_PASSWORD= ********
   POSTGRES_DATABASE= ********
   POSTGRES_URL_NO_SSL= ********
   POSTGRES_PRISMA_URL= ********
   AUTH_SECRET= ********
   AUTH_URL= ********

   ```

4. Run development server:
   ```bash
   npm run dev
   ```

For more information, see the [course curriculum](https://nextjs.org/learn) on the Next.js Website.
